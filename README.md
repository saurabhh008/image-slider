# Image Slider 🎞️

A simple and responsive image slider built using HTML, CSS, and JavaScript.  
It automatically slides images and also has next/prev buttons for manual control.

---

## 🔧 Tech Used

- HTML
- CSS
- JavaScript (Vanilla)

---

## 📸 Features

- Auto sliding images
- Next/Previous buttons
- Responsive design
- Smooth transition effects
